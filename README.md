# Student Grade Analyzer

A Python command-line program that allows a user to input student grades and then calculates useful statistics such as the average, highest, and lowest grade.

## Features
- Accepts multiple grades from the user
- Allows the user to finish input by typing `done`
- Validates input to prevent errors
- Calculates:
  - Average grade
  - Highest grade
  - Lowest grade
- Displays a clean summary output

## Skills Demonstrated
- Python fundamentals
- Loops (`while`)
- Conditional logic (`if / else`)
- Functions
- Lists
- Input validation with `try/except`
- Debugging and program flow control

## How to Run
1. Make sure Python is installed
2. Run the program from the terminal:

```bash
python grade_analyzer.py
